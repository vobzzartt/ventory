# Ventory

Ventory is an AI assistant for your shop. It helps you manage your stock, track your sales, and get insights by talking to your business.

## Features

### Voice Logging
You can log sales and record inventory by talking to the application. It converts your speech to text and updates your records automatically.

### SME Co-Pilot
The AI assistant works as a co-pilot for small and medium enterprises. It answers questions about your business data and helps you make decisions.

### Stock Manager
Track your stock levels in real time. The system alerts you when items are running low and need to be restocked.

### Sales Insights
Get clear reports and insights on your daily sales. Learn which products are performing best and understand your revenue trends.

## Technical Details

The frontend is built using modern web technologies.

* Framework: Next.js with App Router.
* Styling: Tailwind CSS.
* State and Auth: Supabase.
* Animations: Framer Motion.

## Installation

Follow these steps to run the application locally.

1. Install dependencies.
```bash
pnpm install
```

2. Start the development server.
```bash
pnpm dev
```

3. Build the application.
```bash
pnpm build
```

4. Run the production build.
```bash
pnpm start
```
